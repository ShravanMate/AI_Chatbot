## Voice-Controlled AI Chatbot (Python + ML)

This project is a voice-enabled AI chatbot built using Python.
It understands user speech, predicts the intent using a Machine Learning model (TF-IDF + KNN), and responds accordingly.
It can also open Google, open YouTube, and play songs on Spotify through voice commands.

## Features

Voice input using Google Speech Recognition

Intent detection using TF-IDF Vectorizer + KNN Classifier

Supports typed input as fallback

Can open websites (Google, YouTube)

Can search and play songs on Spotify

Customizable dataset (dataset.csv) for adding more intents or replies

Lightweight and easy to run

## Tech Used

Python 3
SpeechRecognition
PyAudio
Pandas, NumPy
Scikit-learn (TF-IDF + KNN)
